### ファイル作成方法
ファイルを作成してアップロードします。
~~~text
  1. ファイル名は任意の名前.jsonで作成
  2. ファイルの内容は作成例を元に作成
~~~

#### 作成例
~~~json
  {
    "title": "たしざん",
    "endText": "おわり",
    "items": [
      { "leftSide": "1+1=", "rightSide": "2" },
      { "leftSide": "1+2=", "rightSide": "3" }
    ]
  }
~~~
